# FlowSpace

A unified productivity workspace combining To-Dos, Notes, and Contacts with powerful CRUD operations, search, and filtering capabilities.

## Tech Stack
- **Frontend**: Next.js 15, React, MUI DataGrid, TailwindCSS
- **Backend**: Node.js, Express, MongoDB
- **Features**: CRUD operations, Pagination, Search, Filtering

## Project Structure
- `apps/frontend` - Next.js application
- `apps/backend` - Express API server
- `packages/shared` - Shared types and utilities

## Getting Started
```bash
# Install dependencies
npm install

# Run backend
cd apps/backend
npm run dev

# Run frontend (in another terminal)
cd apps/frontend
npm run dev
